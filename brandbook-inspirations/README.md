# Brandbook Inspirations

Ce dossier rassemble 10 `DESIGN.md` extraits du dépôt public
[`VoltAgent/awesome-design-md`](https://github.com/VoltAgent/awesome-design-md)
(branche `main`). Chaque fichier est une **analyse éditoriale d'un design system
de marque** : atmosphère visuelle, palette, typographie, composants, rythme de page.

Ils servent de **moodboard textuel** pour le projet **CRI — Cocoa Ranch & Industry**
(agtech / cacao / impact social en Afrique de l'Ouest).

## Notes

- Tous les fichiers ont une frontmatter YAML `version: alpha` et un contenu
  structuré en sections `## Overview`, `## Color`, `## Typography`, etc.
- Aucune image ni binaire n'a été téléchargé (uniquement du Markdown).
- Source unique : `https://raw.githubusercontent.com/VoltAgent/awesome-design-md/main/`
- Le sous-dossier dans le repo source est `design-md/<slug>/DESIGN.md` (et non
  `companies/...` comme suggéré initialement).
- **Linear** vit dans `design-md/linear.app/` — extension conservée dans le nom local.
